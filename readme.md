### Features

- pinyinMaster Chat bot.;
- Goal: a chatbot is able to say hanzi pinyin pronounciation.;
- Language: python3;
- Python modules: telepot, subprocess, sys, pytesseract, PIL;
- Infrastructure: Raspberry pi 4 4GB;
- How to install modules;
**	pip3 install telepot;
**	pip3 install subprocess;
**	pip3 install pytesseract-ocr;
**	pip3 install pytesseract;****
**	pip3 install PIL;**********
